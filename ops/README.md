# Operations & Infrastructure

This directory contains strict configuration for Enterprise Production Deployment.

## Structure
- `docker/`: Docker Compose definitions for Blue/Green deployment.
- `nginx/`: Hardened Reverse Proxy configuration.
- `runbooks/`: Standard Operating Procedures (SOPs) for DR and Security.
- `scripts/`: Operational scripts (Traffic Switch, Migration Gates).
- `secrets/`: Governance rules for secret management.

## Deployment
See `runbooks/` for deployment related procedures.
